02 — The AI layer

Risks that only show up when a model writes the code.

Fifteen documents cover failure modes traditional review misses — from packages that don't exist to security code that only looks like it works.

Supply chain

Hallucinated packages

A model confidently imports a package that doesn't exist — an attacker registers the name and inherits the trust.

Injection

Prompt-injected safeguards

Instructions hidden in context steer the assistant into removing or skipping the checks it was told to add.

Illusion

Fake security code

Output that reads as hardened — validation, escaping, auth — but does nothing at runtime.

Drift

Context-window poisoning

Over a long session the model quietly forgets the security constraints it started with.

Leakage

Prompt / system leaking

System prompts and secrets surfacing through the model's own output.

Upstream

Training-data poisoning

Malicious patterns baked into training data reappear in everyone's generated code.

03 — Real incidents

Eighty-six case studies, each traced to its source.

Every incident was researched from primary sources — NVD, vendor advisories, post-mortems. Where a famous breach has no assigned CVE, it says so.

npm event-stream

A handed-off package hid a wallet stealer, targeting Copay Bitcoin builds.

2020

Poisoned build pipeline reached ~18,000 organizations, including US agencies.

2021

2023

ASP.NET Core request smuggling, CVSS 9.9 — the highest .NET rating to date.

04 — Put it to work

Scan your code, or feed the corpus to your own RAG.

155 Semgrep rules target AI-generated sinks. 1,551 pre-chunked documents carry YAML frontmatter and drop straight into a vector store.

semgrep
# scan a repo with every rule
semgrep --config semgrep-rules/ ./your-project

# or just the framework-sink pack
semgrep --config semgrep-rules/vibe-framework.yml .
embeddings.py
from langchain_community.vectorstores import Chroma
from langchain_community.embeddings import HuggingFaceEmbeddings

emb = HuggingFaceEmbeddings(model_name="BAAI/bge-small-en-v1.5")
db  = Chroma("embeddings/chroma", emb)  # 1,551 chunks
