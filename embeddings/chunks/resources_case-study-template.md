# Case Study Format

Template to use for each case study:

```markdown
# [Case Name] — [Language/Framework]

## 📅 Incident Date
## 🎯 Target System
## 🔴 What Happened?
## 🧠 Root Cause (Root Cause) 
## 💥 Impact
## 🔧 How Was It Resolved?
## 🎓 Lessons Learned
## 🔗 Source / Reference
```
