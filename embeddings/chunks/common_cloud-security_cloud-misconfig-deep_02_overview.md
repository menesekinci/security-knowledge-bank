## Overview

Cloud misconfigurations are the #1 cause of cloud data breaches. This deep dive covers credential leaks in AI-generated code, environment variable exposure, `.env` files committed to git, and the systemic risks of "vibe coding" with AI assistants that generate insecure cloud configurations.
